# 📚 أمثلة الاستخدام

مجموعة من الأمثلة العملية لـ Epic Story Animator

---

## مثال 1️⃣: قصة تاريخية بسيطة

```bash
python main.py "في سنة 1969، هبط الإنسان على القمر. أرملسترونغ والفريق الأمريكي قاموا برحلة تاريخية. كانت أول خطوة إنسان على سطح القمر."
```

**النتيجة:**
- ✅ مشهدين أو ثلاثة
- ✅ صور من NASA
- ✅ فيديو مدة 1-2 دقيقة

## مثال 2️⃣: قصة حضارة قديمة

```bash
python main.py "الحضارة المصرية القديمة استمرت 3000 سنة. بنوا الأهرامات كمقابر ملكية. استخدموا الهندسة المتقدمة والرياضيات. الفراعنة كانوا يحكمون بقوة وحكمة."
```

**ستحصل على:**
- 🗺️ خريطة تاريخية
- 📊 خط زمني
- 🖼️ صور أثرية حقيقية
- 🎬 ترانزيشنات احترافية

## التخصيص المتقدم

### تغيير الإعدادات قبل التشغيل

```python
# في config.py:

# استخدم نموذج أسرع
OLLAMA_MODEL = "mistral"  # بدلاً من llama2

# جودة أقل للتطوير السريع
MANIM_QUALITY = "low_quality"

# فيديو أقصر
MAX_VIDEO_DURATION = 60  # 1 دقيقة

# ألوان مخصصة
PRIMARY_COLOR = "#FF6B6B"      # أحمر
SECONDARY_COLOR = "#4ECDC4"    # تيركواز
ACCENT_COLOR = "#95E1D3"       # نعناع
```

## أمثلة إنتاجية (عالية الجودة)

```bash
# تشغيل مع جودة عالية جداً
# أولاً، عدّل config.py:
# MANIM_QUALITY = "high_quality"
# بعدها:
python main.py "قصتك هنا..."

# الوقت: 30-45 دقيقة
# الحجم: 200-500 MB
# الجودة: 4K cinematography
```

## نصائح لأفضل النتائج

### 1. اكتب قصة مفصلة وواضحة
```python
# ❌ قصير جداً:
"الحرب العالمية الثانية كانت سيئة"

# ✅ أفضل:
"بدأت الحرب العالمية الثانية سنة 1939. ألمانيا النازية توسعت في أوروبا. قاتلت الحلفاء لمدة 6 سنوات. انتهت سنة 1945 بهزيمة ألمانيا واليابان."
```

### 2. استخدم كلمات مفتاح قوية
```python
# كلمات تساعد في البحث عن صور:
"اخترع توماس إديسون المصباح الكهربائي 1879"
# بدلاً من:
"اخترع شخص شيء مهم"
```

### 3. أضف سياق تاريخي
```python
"الثورة الصناعية بدأت في إنجلترا سنة 1760. تطورت الآلات والمصانع. 
غيرت طريقة الإنتاج والعيش. انتشرت لأوروبا وأمريكا."
```

## استكشاف الأخطاء في الأمثلة

### مثال خرج بـ خطأ

```bash
python main.py "قصة بسيطة" 2>&1 | tee error_log.txt
```

بعدها شارك `error_log.txt` لحل المشكلة.
